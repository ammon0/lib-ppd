Object_pt load ( const char *  object_name)

Find the named object on disk and load it and its dependencies into the object memory.

This function searches the current directory for the named class in either dyn text files or pexe object files. It then loads that object into the object memory and adds its imports and exports to the global symbol table. It then recurses on each imported object.
